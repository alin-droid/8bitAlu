# 8-bit ALU in Verilog

This project presents a structural HDL implementation of an 8-bit Arithmetic Logic Unit designed using modular hardware components.

Implemented operations:
- Add
- Subtract
- Multiply
- Divide

The project includes a dedicated Control Unit, datapath registers, arithmetic modules and simulation environment.

## Tools
Verilog HDL, ModelSim
